## 1. 작품 활용 목적 (200자 이내)

> 서울 866개 초·중·고 학교급식 공공데이터를 AI로 분석·시각화하여, 학부모에게는 자녀 학교 급식의 영양·원산지·안전성 정보를, 학생에게는 식생활 인식을, 교사·교육청에는 학교별 급식 운영 진단을 제공한다. 7개 지표 평가, AI 챗봇, 가정-학교 영양 비교, 탄소발자국 분석, 커뮤니티 참여를 통합하여 교육 공공데이터의 실질적 활용 가치를 극대화한다. (199자)

## 2. 배경 및 문제 인식

### 2.1 문제 상황

#### 😟 학부모 측면
- 자녀 학교 급식 메뉴에 대한 **정보 비대칭**
- **알레르기 식품** 여부를 사전에 알기 어려움
- **영양 균형**, **원산지 안전성** 확인 복잡
- 전국 학교 간 급식 품질 **비교 어려움**

#### 😢 학생 측면
- 급식에 대한 **단순 소비자**로만 참여
- **영양 교육** 기회 부족
- 식생활에 대한 **능동적 인식** 결여

#### 😓 교사 측면
- 급식 관련 **학부모 민원 대응** 부담
- **우수 학교 벤치마킹** 경로 부재
- 급식 운영 **객관적 진단** 도구 부족

#### 😔 교육청 측면
- 지역 간 **급식 격차** 실시간 파악 어려움
- **정책 효과 측정** 수단 제한적
- 전국 **통합 현황판** 부재

### 2.2 근본 원인

1. **데이터 개방은 있으나 활용 장벽 높음**
   - NEIS Open API 존재하지만 **일반인 접근성 낮음**
   - 표/텍스트 위주 조회 → **직관적 시각화 부재**

2. **분산된 정보 생태계**
   - 학교별 홈페이지 산재
   - 통합 비교 플랫폼 **전무**

3. **AI 기술 접목 부족**
   - 단순 조회 → **지능형 분석** 미흡
   - 개인화된 해설/추천 **제공 없음**

### 3.1 핵심 아이디어

> **"공공데이터 + AI = 학부모·학생·교사·교육청 통합 플랫폼"**

### 3.2 차별화 포인트

#### 🥇 1위 차별점: 전국 17개 시도교육청 실시간 연동

- ⭐ 즐겨찾기

- 🏆 월간 베스트
- 🎴 카드뉴스 공유
- 🍎 알레르기 필터
- 🤖 AI 챗봇
- 🏠 가정 영양 비교
- 🌱 탄소 발자국
- 📖 데이터 스토리텔링
- 💬 커뮤니티 댓글
- 📱 모바일 반응형
- 🎨 다크 테마
- 🔄 실시간 업데이트

## 4. 주요 기능 상세

### 🗺️ 4.1 지도 탭 - 시각적 탐색
- **17개 시도교육청 드롭다운**으로 지역 전환
- **전국 모드** 선택 시 한반도 전체 11,000+ 학교 표시
- Leaflet.js 기반 인터랙티브 지도
- 학교 유형별 색상 구분 (초록/노랑/파랑)

### 📊 4.2 차트 탭 - 다차원 분석
기본 11개 차트 + **전국 모드 전용 3개 차트**:
- 17개 시도교육청별 학교 수
- 지역별 학교급 Stacked Bar
- 교육청별 직영율 TOP

### 🍱 4.3 목록 & 필터 탭 - 정밀 검색
- **시도교육청 필터** (1차 필터)
- 시/군/구, 학교급, 설립구분, 급식운영, 조리형태 등 **5+단계 필터링**
- 활성 필터 칩으로 조건 한눈에 파악
- CSV 내보내기 지원

### 🏆 4.4 순위 탭 - 지능형 랭킹
- Rule-based 7지표 평가
- **4가지 정렬 방식** (점수↓, 점수↑, 이름순, 교육청별)
- **교육청 배지** 표시 (🏛️ 서울)
- Top 50 전국 순위

### 🤖 4.5 AI 기능
- **Claude API** 자연어 챗봇
- 학교별 AI 해설 자동 생성
- **데이터 스토리텔링** (5챕터 구성)

### 🏠 4.6 가정-학교 영양 비교
- 학부모 가정 식단 vs 학교급식 영양소 비교
- 부족한 영양소 식단 보완 제안

### 🌱 4.7 친환경 탄소 발자국
- 급식의 **탄소 배출량** 계산
- **국내산 vs 수입산** 환경 영향 비교
- **지속가능 교육** 콘텐츠

### 💬 4.8 커뮤니티 참여
- 학교별 좋아요/댓글
- 학부모 참여형 데이터 민주화
- 닉네임 + 역할 선택 시스템
