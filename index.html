• API 호출 비용은 사용자 본인에게 청구됩니다
• Claude Haiku 4.5 모델 사용 (저비용/고속)
• 1회 호출당 약 $0.001~0.003 예상
• 키는 이 브라우저/기기에만 저장됩니다
